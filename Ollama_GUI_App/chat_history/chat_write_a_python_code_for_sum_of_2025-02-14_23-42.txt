This Python program calculates the sum of two numbers.

### Code
```python
def add_numbers(num1, num2):
    """
    Adds two numbers together.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.

    Returns:
        float: The sum of num1 and num2.
    """
    return num1 + num2

# Get the input from user
num1 = float(input("Enter the first number: "))
num2 = float(input("Enter the second number: "))

# Calculate the sum
result = add_numbers(num1, num2)

# Print the result
print(f"The sum of {num1} and {num2} is {result}")
```

### Explanation

*   We define a function called `add_numbers` that takes two parameters, `num1` and `num2`.
*   Inside the function, we return the sum of `num1` and `num2`.
*   We get the input from the user for both numbers using the `input()` function.
*   We calculate the sum by calling the `add_numbers` function with the user inputs as arguments.
*   Finally, we print the result to the console.
