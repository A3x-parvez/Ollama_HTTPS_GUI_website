# Markdown File Demo

This is a basic markdown file demonstrating some common features. 

## Heading Levels

You can use **`#`** for heading levels:

* **Heading 1:** # This is a Heading 1
* **Heading 2:** ## This is a Heading 2

### Emphasis and Lists

* **Emphasis using asterisks or underscores:**  *this text is emphasized*
    * List item 1
    * List item 2

Use **`_`** for a bold emphasis. Use **`***`** to emphasize italicized text.


```python
print("This is some Python code!")

# Output: This is some Python code!
```

### Links and Images

* Link with URL: [Google](https://www.google.com)
* Image with link: ![alt-text](image_url)

### Code Blocks 

You can use triple backticks ( ``` ) to format code:

```python
print("Hello World!")
```
